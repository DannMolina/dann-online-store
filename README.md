# Get started

**Install dependencies**

```
  npm install
```

**Start the app**

```
  npm run start
```

**Open to your browser**

```
http://localhost:3000
```

# Technology used

- Reactjs
- TypeScript
- Ant Design 5.0
- Antd icons
